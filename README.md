# FlexBiz - Premium Business HTML Template

Modern and responsive business website template designed for startups, agencies, freelancers, consultants, SaaS companies, and growing businesses.

Built using HTML5, CSS3, Bootstrap 5.3, and Bootstrap Icons with a clean architecture, responsive layout, and modern SaaS-inspired design.

---

## Overview

FlexBiz is a multi-page business website template that combines modern UI design, responsive layouts, and dark mode support.

The template is suitable for:

* Business Websites
* Agency Websites
* Startup Landing Pages
* Company Profiles
* Freelance Portfolios
* SaaS Product Websites

## Installation

### Method 1 - Open Directly

1. Download the template package.
2. Extract the ZIP file.
3. Open `index.html` in your browser.

### Method 2 - Development Mode

1. Open the project using Visual Studio Code.
2. Install the Live Server extension.
3. Right click `index.html`.
4. Select **Open with Live Server**.

---

## Customization

### Change Brand Name

Search and replace:

```html
FlexBiz
```

inside all HTML files.

### Change Colors

Open:

```text
assets/css/style.css
```

Modify:

```css
:root{
    --primary:#4F46E5;
    --secondary:#7C3AED;
}
```

### Change Images

Replace images inside:

```text
assets/images/
```

or update image URLs directly within HTML files.

### Change Content

Edit content inside:

```text
index.html
about.html
services.html
portfolio.html
pricing.html
faq.html
contact.html
```

## Dark Mode

FlexBiz includes a complete dark mode system.

Features:

* One-click theme switching
* Theme persistence using localStorage
* Dedicated darkmode.css file
* Optimized contrast and readability

User preferences are automatically saved and restored.
